After Install the Ubuntu 15.04 for your computer

Install Node.js, npm and nvm

Install Node.js

                       
    $ sudo apt-get install nodejs

If you intalled sucsessfully you can check it using terminal. Open the terminal and type

                       
    $ nodejs -v
    // you can see result as
    v0.10.25
    // version can be change

You can open nodejs terminal typing nodejs. Ater that type 

                       
    $ nodejs
    > console.log("hello nodejs");
    hello nodejs    // result will give like this

Install Node Package Manager(npm)

Type this command to insall nodejs package manger

                       
    $ sudo apt-get install npm

Install Node Version Manager(nvm)

To start off, we'll need to get the software packages from our Ubuntu repositories that will allow us to build source packages. The nvm script will leverage these tools to build the necessary components:

                       
   $ sudo apt-get update
   $ sudo apt-get install build-essential libssl-dev curl
   $ curl https://raw.githubusercontent.com/creationix/nvm/v0.16.1/install.sh | sh

Last command will download script and run it.It will install the software into a subdirectory of your home directory at ~/.nvm. It will also add the necessary lines to your ~/.profile file to use the file.

To gain access to the nvm functionality, you'll need to log out and log back in again, or you can source the ~/.profile file so that your current session knows about the changes:

                       
    $ source ~/.profile

To find out the versions of Node.js that are available for installation, you can type:

                       
    $ nvm ls-remote

If we want specific version to install, type this cammand

                       
    $ nvm install <version>
    // example => $ nvm insatll 0.12.5

Usually, nvm will switch to use the most recently installed version. If we want we can use specific version using this command

                       
    $ nvm use <version>
    // example => $ nvm use 0.12.5

If you have multiple Node.js versions, you can see what is installed by typing:

                       
    $ nvm ls

If you wish to default one of the versions, you can type:

                       
    $ nvm alias default <version>

This version will be automatically selected when a new session spawns. You can also reference it by the alias like this:

                       
    $ nvm alias default

Each version of Node.js will keep track of its own packages and has npm available to manage these
