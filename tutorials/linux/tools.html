After Install the Ubuntu 15.04 for your computer

Install Oracle java on your computer

Install oracle java 6 using PPA command

                       
    sudo add-apt-repository ppa:webupd8team/java
    sudo apt-get update
    sudo apt-get install oracle-java6-installer

Install oracle java 7 using PPA command

                       
    sudo add-apt-repository ppa:webupd8team/java
    sudo apt-get update
    sudo apt-get install oracle-java7-installer

Install oracle java 8 using PPA command

                       
    sudo add-apt-repository ppa:webupd8team/java
    sudo apt-get update
    sudo apt-get install oracle-java8-installer

You JVM directory is /usr/lib/jvm/java-<version>-oracle/

Now you can add your enviroments variable JAVA_HOME and JRE_HOME. First open the terminal type this command. you will document with PATH varible.

                       
    sudo gedit /etc/enviroments

Add these two lines at the begining and add $JAVA_HOME and $JRE_HOME variables at the end of the PATH. If you want set java_home as java 6 or 7 then add the path for java version.

                       
    JAVA_HOME="/usr/lib/jvm/java-8-oracle"
    JRE_HOME="/usr/lib/jvm/java-8-oracle/jre"


    PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games:$JAVA_HOME:$JRE_HOME"

You can see the java version that you have installed using javac -version command. It will show the latest version you have installed.

Install Open JDK on your computer

Open JDK is another JDK for java. There is no different. But if you want add them using this command

Install Open jdk 6

                       
    sudo apt-get update 
    sudo apt-get install openjdk-6-jdk

Install Open jdk 7

                       
    sudo apt-get update 
    sudo apt-get install openjdk-7-jdk

Install Open jdk 8

                       
    sudo apt-get update 
    sudo apt-get install openjdk-8-jdk
