Servlet & JSP

In this tutorial i'am going to show you how Servlet and JSP work. I used eclips mars as my ide, servlet version 2.5 and tomcat 6.0.41 as web container. Install these tools in your computer

Open the eclips and click File > new > Dynamic Web Project. This window will appear.

Select a name for the project.My project is ServletTest. next Click > New Runtime and select tomcat version as 6. now add the path to tomcat directory and click finish.

Now click the finish again.This is the folder structure of the project that you have created. WebContainer folder keep the all html, JSP, js, css and images in your project.Later i'll explain about web.xml file. Src folder keep the all java files in your project.

Now Right click on the project go to New > HTML file and select a name as index.html, click finish.

Now add this line to your index.html file and save it.

Now again Right on the project and select New > Servlet. After that a new window will pop-up. Now give package name and Servlet name as you Wish, click Next.In this window you can select a path.When you type this path after context path( your project name) This servlet will invoke acording to your request(Get or Post).
