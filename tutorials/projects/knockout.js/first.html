    function myViewModel(){
        this.firstName = ko.observable("");
        this.lastName = ko.observable("");
        this.fullName = ko.computed(function(){
            return this.firstName()+" "+this.lastName();
        },this);

     };

     var myviewmodel = new myViewModel();
     ko.applyBindings(myviewmdel); // activate the Knockout.js

We can bind values from ViewModel to HTMl elements using 'data-bind' attribute in body section.

                        
    FirstName :

ko.observable is a concept which keeps an eye on value changes so that it can update underline ViewModel variable.

                        
    this.firstName = ko.observable("Ajantha");

ko.applyBindings() function is used to activate the knockoutjs.If you want to get value in ViewMode you can access it like this.myViewMode.yourName();

Next thing is, when any changes happen in firstName and lastName that will reflected to fullname, so no need to write aditional code for that.
