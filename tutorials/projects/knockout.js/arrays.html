Obersvable Arrays

In observable arrays we can keep collection of objects. This is very useful wen create complex applications.Observable arrays only track which object is added or removed. It does not notify if individual object's properties are modified.

                        
    this.nameOfArray = ko.observableArray(); // this is an empty array

We can initialize it like this

                        
    var nameOfArray = ["ajantha","bandara"];
    this.arrayName = ko.observableArray(nameOfArray);
    // we can access elements in array 
    alert(arrayName()[index]);

There are many functions in observableArrays. They are same as javascript array functions. All works in every browser. These functions can keep track of dependencies.
