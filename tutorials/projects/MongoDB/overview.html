Drop Database

db.dropDatabase() command is used to drop existing database

Example

> use mydb
switched to db mydb
>
> db.dropDatabase()
{ "dropped" : "mydb", "ok" : 1 }

now check list of databses using show dbs command

> show databases;
contactlist  0.078GB
local        0.078GB

The createCollection() Method

db.createCollection(name, options) command is used to create collection

name is the name of the collection and options is a document and used to specify configuration of collection

Example

> db.createCollection("testCollection")
{ "ok" : 1 }
>

you can see the collection have been created using show collections command

> show collections
system.indexes
testCollection
>

normaly we don't need to create collection, because MongoDB create collection automatically when we insert some document.

> db.elvinlabs.insert({"name":"elvinlabs.com"})
WriteResult({ "nInserted" : 1 })
> show collections
elvinlabs
system.indexes
testCollection
>

The drop() Method

db.collection.drop() method is used to drop specific collection

Example

> show collections
elvinlabs
system.indexes
testCollection
>
> db.testCollection.drop()
true
>
> show collections
elvinlabs
system.indexes
>

Datatype list supports by the MongoDB

  • String : String in MongoDB must be UTF-8 valid and this is most commonly used datatype.
  • Integer : Used to store the numerical value. Integer can be 32bit or 64bit depending upon your server
  • Boolean : Used to store the boolean value (true/false)
  • Double : Used to store floating point value.
  • there are many other datatypes not listed here

RDBMS Where Clause Equivalents in MongoDB

Operation Syntax Example RDBMS Equivalent
Equality {<key>:<value>} db.mydb.find({"writtenBy":
"AjanthaB"}).pretty()		 Where writtenBy="AjanthaB"
Less Than {<key>:{$lt:<value>}} db.mydb.find({"likes":{$lt:50}}).pretty() Where likes < 50
Less Than Equals {<key>:{$lte:<value>}} db.mydb.find({"likes":{$lte:50}}).pretty() Where likes <= 50
Greater Than {<key>:{$gt:<value>}} db.mydb.find({"likes":{$gt:50}}).pretty() Where likes > 50
Greater Than Equals {<key>:{$gte:>value<}} db.mydb.find({"likes":{$gte:50}}).pretty() Where likes >= 50
Not Equals {<key>:{$ne:<value>}} db.mydb.find({"likes":{$ne:50}}).pretty() Where likes != 50

AND operation in MongoDB

To use AND operation with find() method we should use multiple keys seperating with ','

db.mydb.find({key2:value1},{key2:value2}).pretty()
