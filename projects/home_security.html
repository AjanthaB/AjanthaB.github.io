Simple Home Security System

using Intel Galileo Gen 2 Board with image processing

Posted by AjanthaB on June 25, 2015

Safety of the house or office is very important nowadays. So to improve the safty this project may be very helpfull.

Abstract

Our project aims to implement a home security system using the Galileo board and a web camera. This system uses SMS services to notify the user and e-mails a picture taken from the web cam to the user.

The system uses image processing to detect motion. So that the Gallileo board is used at its one of optimum potential.

And moreover this project takes the traditional CC tv camera systems some intelligence so that they would notify the user at a disturbance without just recording all sorts of unnecessary stuff. So at a critical situation the user can taken necessary actions.

System Design and Architecture

The galleo board is a microcontroller which is capable of installing a OS on tp of it using a SD card. Once the OS is installed the board can run phthon codes and even connect to the internet so that it would be enough to implement our project.

So this project mainly focuses on running python and open CV image processing libraries on top of the Galileo board. For this there should be algorithms developed and coded in python for image processing and sending data. And Linux operating system should be installed on the Gallio board usind an SD card. Assembling and testing Hardware components should be done .Finally testing product for all possible test cases.

Methodology & Implementation Details

The Linux operating system was installed to the microcontroller using an SD card the python scripts can be run and the board is given an ip address and can be connected to internet. The system has a separate G-Mail account which only has the contact of the user and only interacts with him

The system periodically checks the last e-mail of the systems inbox. If the last e-mail “ON” the system is activated. If it consists “OFF” the system is deactivated. If it consist of “SEND” or “STOPSEND” the process of emailing images starts or stops

The algorithm for image processing is straight forward. Open CV library is used for image processing. Photos are taken at each second. Then the sum each pixels RGB value is compared with the value of the previous value. If the difference is large there has been a motion taken place.

IMAP library is used to send and receive mail.
Mraa library is used to activate the pins for the buzzer. To activate the buzzer PWM is used.
SMS is send VIA a pre paid gateway site. “textbiz.lk”
Once the system is installed the user can control it using sending an e-mail with “ON” command. At a disturbance the system sends an SMS via the message gateway site. And the buzzer is activated.
By e-mailing the “SEND” command the taken images are emailed to the user. If “OFF” is E-mailed the system stops and the buzzer is deactivated.
